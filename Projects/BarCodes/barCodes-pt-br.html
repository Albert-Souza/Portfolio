BarCodes

Um software que gera e lê código de barras

Execução

Para executar o programa é necessário ter o JRE (Java Runtime Environment) instalado.

  1. Abra um terminal no diretório BarCodes.
  2. Utilize o comando: java BarCodes.

Uso

  • Clique no botão Generate Barcode para gerar um código.
  • Clique em Save para salvar o código no diretório Bars.
  • Clique no botão Read Barcode para abrir um código salvo em seu computador.
